Detailed Description

Function prototypes and structures for controlling HD44780 LCD.

This file contains all definitions, macros, and function prototypes required to initialize and control an HD44780-compatible LCD in 4-bit mode, including support for brightness PWM, keypad input, weight measurement via HX711, and hierarchical menu display.
