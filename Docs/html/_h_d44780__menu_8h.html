Detailed Description

Menu system functions for HD44780 LCD display.

Provides functions to initialize, navigate, and refresh hierarchical menus on the HD44780 LCD module. Supports next/previous item navigation, enter/back actions, and retrieving current menu level and index.

This header defines the interface for menu management in STM32-based projects with HD44780 displays.
