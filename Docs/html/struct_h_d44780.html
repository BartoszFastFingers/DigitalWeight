Detailed Description

Structure representing an HD44780 LCD display instance.

This structure holds all the necessary pins, buffers, and control structures to interface with an HD44780-based LCD in 4-bit mode. It also contains menu display state and control flags for brightness, weight sensor (HX711), and keypad input.
