\doxysubsection{Detailed Description}
Callback functions for \doxylink{struct_h_d44780}{HD44780} LCD display and menu control. 

Provides a set of callback functions for\+:
\begin{DoxyItemize}
\item Displaying weight and probe count
\item Adjusting display brightness
\item Setting custom offset values
\item Handling menu navigation and state machine
\end{DoxyItemize}

These functions interact with the \doxylink{struct_h_d44780}{HD44780} buffer, keypad input, and \doxylink{struct_h_x711}{HX711} weight sensor to manage real-\/time display updates and menu interactions on STM32-\/based projects.
