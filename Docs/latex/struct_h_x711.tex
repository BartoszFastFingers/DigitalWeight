\doxysubsection{Detailed Description}
Structure representing an \doxylink{struct_h_x711}{HX711} load cell amplifier. 

This structure holds the GPIO configuration for the data and clock pins, as well as calibration and last read values for the \doxylink{struct_h_x711}{HX711} module.
