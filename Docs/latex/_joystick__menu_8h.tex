\doxysubsection{Detailed Description}
Menu navigation handling using a joystick for \doxylink{struct_h_d44780}{HD44780} LCD. 

Provides function prototypes for mapping joystick inputs (up, down, left, right, click) to menu navigation callbacks on the \doxylink{struct_h_d44780}{HD44780} LCD display.
