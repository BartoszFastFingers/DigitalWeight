\doxysubsection{Detailed Description}
General-\/purpose STM32 library for GPIO, timers, PWM, and menu handling. 

Provides utility structures and functions for\+:
\begin{DoxyItemize}
\item Timer initialization and microsecond delays
\item GPIO configuration and state changes
\item PWM channel management
\item Menu system with hierarchical navigation
\end{DoxyItemize}

This library centralizes common hardware and UI functionalities for STM32-\/based projects.
