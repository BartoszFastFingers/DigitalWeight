\doxysubsection{Detailed Description}
Structure for LCD control and menu interaction. 

This structure combines menu state, PWM backlight control, weight sensor reference, keypad input, and probe/sample configuration to manage the \doxylink{struct_h_d44780}{HD44780} display in STM32 projects.
