\doxysubsection{Detailed Description}
Buffered interface for \doxylink{struct_h_d44780}{HD44780} LCD. 

This file provides function prototypes for writing to an \doxylink{struct_h_d44780}{HD44780} LCD using an internal buffer. Functions include cursor control, character and string writing, partial clearing, and synchronous/asynchronous refresh.
