Reads a single raw measurement from the \doxylink{struct_h_x711}{HX711} ADC. 

This function performs a blocking read of the \doxylink{struct_h_x711}{HX711} data. It first checks if the \doxylink{struct_h_x711}{HX711} is ready using {\ttfamily \doxylink{_h_x711__pro_8c_a55b66fa8373246c0ba2887eaace89034}{is\+Ready()}}. If the device is not ready, it returns the special value {\ttfamily IS\+\_\+\+NOT\+\_\+\+READY}.

The function reads 3 bytes from the \doxylink{struct_h_x711}{HX711}, applies the configured gain, and converts the 24-\/bit two\textquotesingle{}s complement value into a signed long.
